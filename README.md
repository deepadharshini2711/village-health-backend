# Village Health Backend

Backend system for village health support in rural areas.

## Project Description
This project is a backend system to manage healthcare support in villages. 
It allows:
- Adding volunteers, patients, and pharmacies
- Assigning volunteers to patients
- Requesting and confirming medicine orders
- Tracking actions in usage logs

This is useful for remote healthcare management and monitoring medicine distribution.

## Features
1. **Add Volunteers, Patients, and Pharmacies**
   - Stores information like names, contact, and location.
2. **Assign Volunteers**
   - Assign volunteers to patients who need support.
3. **Medicine Request**
   - Patients can request medicines from nearby pharmacies.
4. **Confirm Orders**
   - Updates pharmacy stock after confirming orders.
5. **Usage Logs**
   - Tracks actions like volunteer assignment and medicine order.

## Tech Stack
- **Python 3**  
- **Flask** for API backend  
- **SQLite3** for database
